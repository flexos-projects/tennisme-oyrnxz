# Component Patterns

## Sections

| Section | Classes | Purpose |
|---------|---------|---------|
| Layout | `.container`, `.app-shell`, `.app-content`, `.page-header` | Structural wrappers |
| Sidebar | `.sidebar`, `.sidebar-header`, `.sidebar-nav`, `.sidebar-footer` | App sidebar shell |
| Navigation | `.nav`, `.nav-item`, `.nav-item-icon` | Sidebar + general nav |
| Navbar | `.navbar`, `.navbar-brand`, `.navbar-actions` | Top bar (marketing/public) |
| Buttons | `.btn`, `.btn-primary`, `.btn-secondary`, `.btn-ghost`, `.btn-accent`, `.btn-error` | All button variants + sizes |
| Cards | `.card`, `.card-body`, `.card-hover`, `.card-accent`, `.card-header`, `.card-footer` | Card containers |
| Stats | `.stats-bar`, `.stat-card`, `.stat-label`, `.stat-value`, `.stat-sub` | Metric display |
| Badges | `.badge`, `.badge-primary`, `.badge-accent`, `.badge-success`, `.badge-warning`, `.badge-error`, `.badge-info`, `.badge-neutral` | Status indicators |
| Forms | `.input`, `.select`, `.textarea`, `.form-label`, `.form-group`, `.form-hint`, `.form-error` | Form controls |
| Tables | `.table-wrapper`, `.table` | Data tables |
| Tabs | `.tab-bar`, `.tab` | Tab navigation |
| Modal | `.modal-backdrop`, `.modal`, `.modal-header`, `.modal-body`, `.modal-footer` | Dialog overlay |
| Toast | `.toast-container`, `.toast`, `.toast-success`, `.toast-error`, `.toast-warning`, `.toast-info` | Notifications |
| Empty state | `.empty-state`, `.empty-icon`, `.empty-title`, `.empty-desc` | No-data views |
| Progress | `.progress`, `.progress-bar` | Progress indicators |
| Grid | `.grid`, `.grid-2`, `.grid-3`, `.grid-4` | Responsive grid layouts |
| Flex | `.flex`, `.flex-col`, `.flex-wrap`, `.items-center`, `.justify-between` | Flex utilities |
| Text | `.text-xs` through `.text-3xl`, `.text-muted`, `.text-secondary`, `.text-primary`, `.text-accent` | Typography |
| Theme | `.theme-toggle` | Dark/light toggle button |

## Key Principles

- Zero hardcoded values — everything via `var()`
- Responsive: grids collapse on mobile, touch targets enforced at 44px
- Hover states wrapped in `@media (hover: hover)` — no hover on touch
- Reduced motion respected via `@media (prefers-reduced-motion)`
- iOS zoom prevented with `font-size: 16px` on mobile inputs
