# Design Tokens

## Default Theme: FlexOS Dark

Indigo primary on a cool dark background. Developer-grade, clean, professional.

- **Primary:** Indigo (hue 265) — confident, modern, tech-forward
- **Accent:** Teal (hue 190) — complementary, used for secondary actions
- **Surfaces:** Cool grey with subtle blue undertone
- **Text:** High-contrast white on dark, with three semantic levels

Dark mode is default. Light mode available via `data-theme="light"` on `<html>`.
All colors in OKLCH for perceptual uniformity and easy theme generation.
