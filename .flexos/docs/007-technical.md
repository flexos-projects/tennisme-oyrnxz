# Technical

## Stack

- **Framework:** Nuxt 4 (Vue 3, Nitro server)
- **State:** Pinia
- **Styling:** Custom design tokens (CSS custom properties) + structural component classes
- **Deploy:** Vercel (auto-deploy on push to main)
- **No auto-imports** — all imports are explicit

## What Ships Pre-Built

The template includes a working Nuxt 4 portal app with:

- Portal page (`app/pages/index.vue`) — level-aware, reads project.json
- 4 portal components — ProjectHeader, DesignSystemPreview, PrototypeNavigator, BuildProgress
- App shell layout (`app/layouts/app.vue`) — sidebar + topbar stub
- Composables — useProject (reactive project.json), useFlexOS (level/status helpers)
- UI store (Pinia) — modal/toast/sidebar state
- Server route — serves `.flexos/prototype/*` files at `/preview/*`
- CSS — tokens.css + components.css loaded globally via main.css

## What Gets Added by Build

- `app/pages/app/` — authenticated app screens
- `app/components/app/` — app-specific components
- `composables/useAuth.ts` — authentication
- `server/api/` — API routes
- Database layer (Convex, Supabase, etc. depending on specs)
